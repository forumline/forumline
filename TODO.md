## Supabase Setup (when at a computer)

1. Login to Supabase CLI:
   ```bash
   supabase login
   ```

2. Run the automated setup script:
   ```bash
   ./scripts/setup-supabase.sh
   ```

This will create the project, apply schema, and configure GitHub secrets automatically.
